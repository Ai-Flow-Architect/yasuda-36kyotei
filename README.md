# 36協定自動化ツール

社労士事務所向けの36協定書自動生成・メール送信ツール。
Excelの顧客データから、5種類の様式パターンに対応したWord協定書を自動生成し、メールで一括送信します。

## セットアップ

### 必要環境
- Python 3.9以上
- Windows 10/11

### インストール

```bash
# 方法1: setup.bat をダブルクリック（Windows）

# 方法2: コマンドライン
pip install -r requirements.txt
```

## 使い方

### 基本操作（Word生成のみ・メール送信なし）
```bash
python main.py input.xlsx
```

### メール本番送信
```bash
python main.py input.xlsx --send
```
※ 送信前に確認プロンプト（y/N）が表示されます。

### Windows: run.bat をダブルクリック
`input.xlsx` を同じフォルダに配置して `run.bat` を実行してください。

## 設定（config.json）

| キー | 説明 | 必須 |
|------|------|------|
| 差出人名 | メール署名の差出人名 | ○ |
| 差出人所属 | メール署名の所属 | ○ |
| 差出人電話 | メール署名の電話番号 | - |
| smtp_server | SMTPサーバー | ○ |
| smtp_port | SMTPポート | ○ |
| smtp_user | SMTP認証ユーザー | ○ |
| smtp_password | SMTP認証パスワード（※環境変数推奨） | ○ |
| from_address | 送信元メールアドレス | ○ |

### セキュリティに関する注意
SMTP認証情報は**環境変数**での設定を推奨します：
```bash
set SMTP_USER=your@email.com
set SMTP_PASSWORD=your_password
set SMTP_FROM=your@email.com
```
環境変数が設定されている場合、config.jsonより優先されます。

## Excel入力ファイルの構造

43列（A〜AQ）+ メールアドレス列（AR）

### 主要な列

| 列 | 項目名 | 入力形式 | 必須 |
|----|--------|---------|------|
| D | 事業所名 | テキスト | ○ |
| E | 事業主名 | テキスト | ○ |
| G | 事業の種類 | テキスト | ○ |
| I | 業務の種類① | テキスト | ○ |
| J | 労働者数 | 数値 | ○ |
| M | 延長時間/1日 | 数値（時間） | ○ |
| N | 延長時間/1ヶ月 | 数値（時間） | ○ |
| V | 特別条項の有無 | 「あり」or「□」 | ○ |
| AR | 送信先メールアドレス | メールアドレス | - |

※ 全43列の詳細は `excel_reader.py` の `COLUMN_MAP` を参照

## 対応様式（5パターン）

| 様式 | 用途 | 自動判定条件 |
|------|------|-------------|
| 様式第9号 | 一般条項（月45h以内） | デフォルト |
| 様式第9号の2 | 特別条項付き（月45h超） | V列「あり」or W〜AD列入力あり |
| 様式第9号の3 | 研究開発業務 | I列に「研究」「新技術」「新商品」 |
| 様式第9号の4 | 適用猶予事業 | G/I列に「運転」「建設」「医師」等 |
| 様式第9号の5 | 適用猶予＋事業場外みなし | 9号の4条件＋「事業場外」「みなし」 |

## 出力ファイル

- **Word協定書**: `output/36協定書_{事業所名}_{様式名}.docx`
- **送信ログ**: `output/send_log_{日時}.csv`

## エラーメッセージと対処法

| エラー | 原因 | 対処法 |
|--------|------|--------|
| 対応していないファイル形式 | .xlsx以外のファイル | .xlsx形式で保存し直す |
| 「{項目}」が未入力です | 必須項目が空 | Excelの該当セルを入力 |
| 「{項目}」が数値ではありません | 数値欄にテキスト | 数値のみ入力 |
| 月{N}時間は45時間超ですが特別条項が未設定 | V列未設定 | V列に「あり」を入力 |
| Word生成失敗 | データ不整合 | ログで対象行を確認し修正 |
| SMTP接続失敗 | ネットワーク/認証エラー | サーバー設定・認証情報を確認 |
